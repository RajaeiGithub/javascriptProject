کار با متغیرها در جاواسکریپت

طور کلی متغیر یا Variable یک ظرف برای نگهداری مقادیر داده ها می باشد ، یعنی به راحتی میتونیم داده هامون رو بریزیم تو این ظرف ها بعد مثلا پردازش مورد نظرمون رو روشون اعمال کنیم و در نهایت به خروجی بفرستیمشون.

حالا نه فقط در جاواسکریپت ، بلکه در تمامی زبان های برنامه نویسی نیز میشه متغیر یا Variable تعریف کرد و در کل میتوان همانند ظرفی برای نگهداری مقادیر ازشون استفاده کرد.

در جاواسکریپت ما با استفاده از ۳ کلمه کلیدی var و let و const میتونیم متغیرها رو تعریف کنیم.

تا قبل از سال ۲۰۱۵ ، تنها روشی که میشد باهاش متغیرها رو در جاواسکریپت تعریف کرد استفاده از کلمه کلیدی var بود.

با انتشار ورژن یا نسخه ۲۰۱۵ زبان جاواسکریپت که با نام رسمی ES6 – ECMAScript 2015 دو تا کلمه کلیدی بنام های let و const هم منتشر شد.

قوانین انتخاب نام متغیرها

باید نام یک متغیر ( Unique Name ) باشه ، یعنی منظورش اینه که نام ها باید منحصربفرد باشند ، یعنی اگه نامی برای یک متغیر تعریف میکنیم باید اون نام در تمام برنامه همون یکی باشه و نباید نام متغیرها شبیه هم باشن.

در مثال های بالا خب دیدیم که ما مثلا متغیرهایی بنام a و b و c به راحتی تعریف کردیم و این نشون میده که نام متغیر میتونه یه نام دلخواه خودتون باشه و حتما اینجوری نیست که مثلا باید a باشه.

ولی خب همین نام دلخواه یسری قوانین داره که باید رعایت کنیم ، این قوانین به شرح زیر می باشد :

  1. نام متغیر باید از حروف کوچک یا بزرگ انگلیسی ( a-z یا A-Z ) باشد ( میتونه هم ترکیبی باشه )
  2. نام متغیر میتونه شامل اعداد ( ۰ تا ۹ ) باشد، ولی اعداد نباید در ابتدای نام متغیر باشند
  3. نام متغیر میتونه با علامت _ ( زیر خط یا Underscore ) شروع بشه
  4. نام متغیر میتونه با علامت $ ( دلار ) شروع بشه
  5. نام متغیر به حروف کوچک و بزرگ حساس می باشد. ( یعنی A با a براش متفاوته )
  6. نام متغیر نمیتونه از کلمات کلیدی یا کلمات رزرو شده خوده جاوااسکریپت باشه

تفاوت های بین Var و Let و Const

خب هرسه تاش که میدونیم برای تعریف متغیر و نگهداری داده ها هستن و از این بابت هیچ فرقی ندارند نحوه تعریف و مقدار دهیشان هم تقریبا یکی است ولی خب یسری تفاوت هایی در نحوه استفاده دارند که در ادامه باهاشون آشنا خواهید شد :

  1. متغیر Var رو میشه دوباره تعریف کرد و مقدار جدید بهش داد
  2. متغیر Var رو میشه اول بهش مقدار داد بعد تعریفش کرد
  3. متغیر Let رو نمیشه دوباره تعریف کرد و مقدار جدید بهش داد
  4. متغیر Let رو نمیشه اول بهش مقدار داد بعدش تعریفش کرد
  5. مقدار متغیر Const همواره ثابت هستش و در داخل برنامه قابل تغییر نیست
  6. متغیر Const رو نمیشه دوباره تعریف کرد و مقدار جدید بهش داد
  7. متغیر Const رو نمیشه اول بهش مقدار داد بعدش تعریفش کرد
